5. ¿Cómo obtener un reporte de Jornadas Laborales?¶

El Reporte de Jornadas Laborales muestra las jornadas laborales de un empleado o de un grupo en determinados días según el fichaje realizado por cada uno de ellos.

  • Seleccionar la opción de menú Personal ‣ Reporte de jornadas.

Se abrirá la pantalla Reporte de Jornadas Laborales.

El reporte tiene varios parámetros.

Pantalla Reporte de Jornada Laboral

  1. El estilo del reporte es ‘Jornadas Laborales’ .

  2. Seleccionar cómo se van a ordenar los datos en la primer lista desplegable Opción. Se puede ordenar ‘Por empleado’ o ‘Por empleado agrupado por día’

  3. Ingresar el rango de fechas para el cual se quieren obtener las jornadas laborales. (Ver Ingreso de rangos de fecha).

  4. Seleccionar la sucursal para la cual se quiere obtener el reporte.

  5. El reporte puede acotarse a un conjunto de empleados, realizando el reporte para un determinado Filtro.

    Para que el reporte sea filtrado, presionar el botón . Se abrirá la pantalla de ingreso de filtros de la búsqueda de empleados. Ingresar el criterio de búsqueda y presionar el botón Aceptar. En el campo inferior al botón se muestra la descripción del criterio de búsqueda a aplicar.

  6. Presionar el botón Aceptar para generar el reporte.

  7. Seleccionar la salida del reporte. El reporte se puede obtener por pantalla, impresora, etc. Si sale por pantalla, posteriormente puede imprimirse sin tener que volver a completar los parámetros del reporte ni ejecutarlo nuevamente.
